# 📦 Multi ZIP File Extractor

A simple and efficient Python tool to extract multiple ZIP files at once using a clean file selection interface. Each ZIP file is automatically extracted into its own folder with clear console output, error handling, and enhanced user experience using tqdm and colorama.

---

## 🚀 Features

- 📁 Select multiple ZIP files at once  
- 📦 Extract each ZIP into its own folder  
- ⚡ Fast and lightweight  
- 🎯 Simple GUI file picker (Tkinter)  
- 🛡️ Error handling for corrupted files  
- 🎨 Colored console output (colorama)  
- 📊 Progress indication support (tqdm)  
- ⌨️ Press ENTER to exit  

---

## 🛠️ Requirements

- Python 3.6 or higher

## ⚙️ How It Works

1. Opens a file dialog to select multiple ZIP files  
2. Loops through each selected file  
3. Creates a folder with the same name as the ZIP  
4. Extracts all contents into that folder  
5. Displays status messages in the console  

---

## 💡 Example

If you select:

file1.zip  
file2.zip  

Output will be:

file1/  
file2/
