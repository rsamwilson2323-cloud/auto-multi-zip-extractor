# 📦 Multi ZIP File Extractor

A simple and efficient Python tool to extract multiple ZIP files at once using a clean file selection interface. Each ZIP file is automatically extracted into its own folder with clear console output, error handling, and enhanced user experience using tqdm and colorama.

---

## 🚀 Features

- 📁 Select multiple ZIP files at once  
- 📦 Extract each ZIP into its own folder  
- ⚡ Fast and lightweight  
- 🎯 Simple GUI file picker (Tkinter)  
- 🛡️ Error handling for corrupted files  
- 🎨 Colored console output (colorama)  
- 📊 Progress indication support (tqdm)  
- ⌨️ Press ENTER to exit  

---

## 🛠️ Requirements

- Python 3.6 or higher  
- Git installed  

Required Libraries:
- tqdm  
- colorama  

---

## 📥 Complete Installation Guide

1. Clone the Repository

   git clone https://github.com/rsamwilson2323-cloud/auto-multi-zip-extractor.git

2. Navigate to Project Folder

   cd auto-multi-zip-extractor

---

## 🧪 Optional Virtual Environment (Recommended)

Windows:

   python -m venv venv  
   venv\Scripts\activate  

Linux / macOS:

   python3 -m venv venv  
   source venv/bin/activate  

---

## 📦 Install Dependencies

   pip install -r requirements.txt  

If pip fails:

   python -m pip install --upgrade pip  
   pip install -r requirements.txt  

---

## ▶️ How to Run

   python "multi zip extractor.py"  

If that doesn't work:

   python3 "multi zip extractor.py"  

---

## 🖥️ How to Use

1. Run the script  
2. File dialog opens  
3. Select multiple ZIP files  
4. Click Open  
5. Files extract automatically  
6. Output folders are created

## ❗ Troubleshooting

Python not found:

   python --version  

pip not working:

   python -m ensurepip --upgrade  

Tkinter error:

Ubuntu:

   sudo apt-get install python3-tk  

Windows:

Reinstall Python with Tkinter enabled  

Permission error:

   sudo python "multi zip extractor.py"
